# 血液ガス・酸塩基平衡 評価アルゴリズム仕様書 (v2.6.3用)

本仕様書は、KetsuekiGasアプリ (v2.6.3) の計算エンジン（`src/engine.ts`）で実行される**すべてのアルゴリズム（入力から出力への加工・判定フロー）**を網羅的に定義するものです。

## 全体フロー
入力データは以下の順序で評価・加工され、最終的な鑑別結果として出力されます。

1. **静脈血(VBG)補正** (静脈血選択時のみ)
2. **酸素化の評価** (PaO2, FiO2, PaCO2)
3. **Step 1: pHの評価** (アシデミア / アルカレミア)
4. **Step 2: 一次性変化の評価** (pH, PaCO2, HCO3(あるいはBE))
5. **Step 3: 呼吸性・代謝性代償の評価** (予想PaCO2 / 予想HCO3の計算)
6. **Step 4: アニオンギャップ(AG)の評価** (AG開大 / 隠れHAGMAの判定)
7. **Step 5: 隠れた混合性障害・合併の評価** (補正HCO3 / ΔAG)
8. **最終鑑別 (Differentials)** (各ステップ結果と尿検査値からの原因疾患の絞り込み)

---

## 1. 静脈血 (VBG) の動脈血 (ABG) への変換補正
ユーザーが「静脈血 (Venous)」を選択した場合、計算エンジンに入る前に以下の通り近似式を用いて動脈血相当の値へ補正します。
- **pH**: `静脈血pH + 0.035`
- **PaCO₂**: `静脈血pCO2 - 7.5` mmHg
- **HCO₃⁻ / BE**: `静脈血HCO3 (又はBE) - 2.0` mEq/L
※ **静脈血の酸素化評価の無効化**: VBGのpO₂は循環状態や末梢組織の酸素消費量に大きく左右されるため、正確なP/F比やA-aDO₂算定には使用できません。そのため静脈血選択時は酸素化の評価（以下2.の全プロセス）をスキップし、「静脈血(VBG)酸素化評価不可」として警告します。

---

## 2. 酸素化の評価
PaO₂（および任意でFiO₂, PaCO₂）が入力された場合、以下の指標を計算し評価します。

1. **P/F比 (PaO₂/FiO₂)**
   - 計算式: `PaO₂ / (FiO₂ % / 100)`  ※FiO₂未入力時は室内気(21%)として計算
   - **< 300**: ⚠️ 急性肺障害等の酸素化低下リスクあり（ARDS基準等）
2. **A-aDO₂ (推定肺胞気動脈血酸素分圧較差)**
   - 計算式 (室内気1気圧を想定した簡易式): `(713 × FiO₂ - PaCO₂ / 0.8) - PaO₂`
3. **加齢によるPaO₂の生理的低下の評価 (年齢入力時)**
   - 計算式 (仰臥位予測): `予想PaO₂ = 104 - 0.27 × 年齢`
4. **低酸素血症の判定**
   - **PaO₂ < 60 Torr**: 🚨 急性の低酸素血症（呼吸不全）として直ちに警告。
   - **60 ≦ PaO₂ < 80 Torr**:
     - 年齢相応の予測値（許容範囲-5 Torr程度）を満たす場合: **正常 (加齢考慮)**（加齢に伴う生理的な低下）
     - 予測値を満たさない場合 または 年齢未入力時: **⚠️ 軽度低酸素**
   - **PaO₂ ≧ 80 Torr**: 正常な酸素化。

---

## 3. Step 1: pHの評価 (Acidemia / Alkalemia)
- **アシデミア**: `pH < 7.35`
- **正常範囲**: `7.35 ≦ pH ≦ 7.45` （※混合性障害の可能性は残る）
- **アルカレミア**: `pH > 7.45`
- ※ `pH ≦ 7.15` または `pH ≧ 7.60` の場合は緊急対応が必要(Urgent)としてフラグを立てる。

---

## 4. Step 2: 一次性変化の評価 (Primary Disorder)
pHと、PaCO2（呼吸性因子）、BEまたはHCO3（代謝性因子）の逸脱方向から、主要な異常の起因を判定します。

- **混合性アシドーシス**: `pH < 7.40` かつ `PaCO₂ > 45` かつ `BE < -2`
- **呼吸性アシドーシス**: `pH < 7.40` かつ `PaCO₂ > 45`
  - 急性/慢性の判定: `BE > 3` なら慢性 (代償が進んでいる)、それ以外は急性。
- **代謝性アシドーシス**: `pH < 7.40` かつ `BE < -2`

- **混合性アルカローシス**: `pH ≧ 7.40` かつ `PaCO₂ < 35` かつ `BE > 2`
- **呼吸性アルカローシス**: `pH ≧ 7.40` かつ `PaCO₂ < 35`
  - 急性/慢性の判定: `BE < -3` なら慢性、それ以外は急性。
  - ※ただし、`BE < -5` など著明な低下がある場合は「代謝性アシドーシス＋呼吸性アルカローシスの混合（例：サリチル酸中毒）」として扱う。
- **代謝性アルカローシス**: `pH ≧ 7.40` かつ `BE > 2`

---

## 5. Step 3: 代償の評価 (Compensation)
一次性変化に対する生体の代償機能が適切に働いているか（予想される範囲内か）を計算し、逸脱があれば別の障害の「合併」を判定します。

1. **代謝性アシドーシス時** (Winterの式)
   - 予想PaCO₂ = `1.5 × HCO₃⁻ + 8` (±2 mmHg)
   - 予想より**低い**: 呼吸性アルカローシスの合併
   - 予想より**高い**: 呼吸性アシドーシスの合併

2. **代謝性アルカローシス時**
   - 予想PaCO₂ = `HCO₃⁻ + 15` (±2 mmHg、上限60)
   - 予想から逸脱している場合は同様に呼吸性障害の合併と判定。

3. **呼吸性アシドーシス時**
   - ΔPaCO₂ = `実測PaCO₂ - 40`
   - [急性] 予想HCO₃⁻ = `24 + 0.1 × ΔPaCO₂` (±2.5)
   - [慢性] 予想HCO₃⁻ = `24 + 0.4 × ΔPaCO₂` (±2.5)

4. **呼吸性アルカローシス時**
   - ΔPaCO₂ = `40 - 実測PaCO₂`
   - [急性] 予想HCO₃⁻ = `24 - 0.2 × ΔPaCO₂` (±2.5)
   - [慢性] 予想HCO₃⁻ = `24 - 0.5 × ΔPaCO₂` (±2.5)

---

## 6. Step 4: アニオンギャップ (AG) の評価
1. **AGの計算**: `AG = Na - (Cl + HCO3)`
2. **Albumin補正**: `補正AG = AG + (4.0 - Alb) × 2.5` （Alb入力がない場合は4.0として計算）
3. **判定**: 補正AG > 12 を「AG開大（高AG性代謝性アシドーシス: HAGMA）」とする。
4. **隠れHAGMA判定**:
   - 代謝性アシドーシス系において AG ≦ 12 であっても、`Lac ≥ 2.5` または `Glu > 250` が存在する場合は「高Cl血症等によりAGが見かけ上正常になっている」と判断し、強制的にHAGMAフローへ移行させる。

---

## 7. Step 5: 隠れた混合性障害の評価 (Delta Ratio & 補正HCO3)
AG開大性アシドーシス (HAGMA) と判定された場合、他の酸塩基平衡異常が隠れていないかを評価するために「ΔAG / ΔHCO3 (Delta Ratio)」を算出し判定します。

- `Delta Ratio = (補正AG - 12) / (24 - 実測HCO3)`
- **< 0.4**: AG正常型代謝性アシドーシス (NAGMA) の合併（下痢・RTA等）を強く疑う
- **0.4 〜 1.0**: HAGMAとNAGMAの混合
- **1.0 〜 2.0**: 合併なし (純粋なHAGMA単独。典型的には乳酸約1.6、ケトン約1.0)
- **> 2.0**: 代謝性アルカローシスの合併（嘔吐・利尿薬等）

（※必要に応じて `補正HCO3 = 実測HCO3 + (補正AG - 12)` も表示し、22〜26を正常範囲として合併診断の補助とします。）

---

## 8. 最終鑑別 (Differentials) の詳細フロー
上記のStep 1〜5の結果に加え、病歴や尿検査値から具体的な疾患名を絞り込みます（図4-31等の医学的フロー準拠）。

### 8.1 高AG性代謝性アシドーシス (HAGMA)
以下を順にスクリーニングします。
1. **腎不全** (BUN高値など) → 腎・尿毒症性アシドーシス
2. **乳酸** (`Lac ≥ 2.0`) → 乳酸アシドーシス (敗血症・ショック等)
3. **ケトン体/血糖** (`Glu > 250`) → ケトアシドーシス (DKA, AKA等)
4. **すべて陰性（原因不明のAG開大）の場合**: 血清浸透圧ギャップ(OG)を確認。
   - `血清OG = 実測血清浸透圧 - (2 × Na + BUN / 2.8 + Glu / 18)`
   - **血清OG > 10**: 毒性アルコール中毒（メタノール、エチレングリコール等）
   - **血清OG 正常**: その他の原因（アセトアミノフェン、サリチル酸、D-乳酸等）

### 8.2 正AG性代謝性アシドーシス (NAGMA / 高Cl性)
尿検査値を用いてさらに鑑別を進めます。
1. **尿アニオンギャップ (U-AG)**
   - `U-AG = 尿Na + 尿K - 尿Cl`
   - **U-AG < 0**: アンモニウム排泄正常 → **消化管由来**（下痢、腸管ドレナージ等）
   - **U-AG > 0**: アンモニウム排泄低下等の異常 → 腎由来の疑い（次へ）
2. **尿浸透圧ギャップ (U-OG)** (U-AG > 0 の場合)
   - `尿OG = 実測尿浸透圧 - (2 × (尿Na + 尿K) + 尿素窒素 / 2.8 + 尿糖 / 18)`
   - **尿OG > 150 mOsm/L**: 未測定陰イオン排泄増多 → トルエン中毒、DKA回復期、D-乳酸
   - **尿OG ≦ 150 mOsm/L**: アンモニウム排泄真の低下 → RTA、保存期CKD（次へ）
3. **尿pH (uPH)** によるRTAの鑑別補助
   - アシデミアにも関わらず **尿pH > 6.0**: 遠位RTA (1型) の疑い
   - **尿pH ≦ 6.0**: 近位RTA (2型) または保存期CKD等の疑い

### 8.3 代謝性アルカローシス
- 尿中塩化物イオン (U-Cl) を基準として鑑別します。
- **U-Cl < 20**: 細胞外液量低下 / 食塩反応性（頻度高：嘔吐、利尿薬、胃液吸引）。生理食塩水が有効。
- **U-Cl ≧ 20**: 細胞外液量正常〜増加 / 食塩不応性（原発性アルドステロン症等）。血圧の高低でさらに鑑別。

## 9. 参考・引用文献 (References)
本アルゴリズムは、以下の標準的な医学書および集中治療ガイドラインに基づき設計・アップデートされています。

1. **全体フロー・鑑別アルゴリズム（図4-31等のテキスト画像）**
   - 山川一馬 著『レジデントのための血液ガスハンター』（中外医学社）などの国内標準的な臨床・救急マニュアル。アニオンギャップのAlb補正や、NAGMAのU-AG/U-OG/尿pHを用いた鑑別フローのベースとしています。
2. **代謝性アシドーシスの呼吸性代償（Winter's 式）**
   - Albert MS, Dell RB, Winters RW. *Quantitative displacement of acid-base equilibrium in metabolic acidosis.* Ann Intern Med. 1967;66(2):312-322.
3. **隠れた混合性障害の評価（Delta Ratio / Δ比）**
   - Wrenn K. *The delta (delta) gap: an approach to mixed acid-base disorders.* Ann Emerg Med. 1990;19(11):1310-1313.
   - 補正HCO3およびΔ比のカットオフ（<0.4, 0.4-1.0, 1.0-2.0, >2.0）の標準的な評価基準。
4. **尿浸透圧ギャップ（U-OG / Urine Osmolar Gap）のカットオフ**
   - 集中治療・腎臓内科領域におけるNAGMA鑑別指標（UAGの限界とUOGへの移行、閾値150 mOsm/kg等）に関する最新のリファレンスやUpToDateの知見。
5. **隠れHAGMAと乳酸（Lactate）の評価**
   - *Surviving Sepsis Campaign Guidelines* 等に基づく、敗血症性ショック・循環不全における乳酸値（Lac ≥ 2.0 mmol/L）のクライテリア。
6. **VBG（静脈血ガス）の限界と動脈血（ABG）との解離**
   - 循環不全・ショック状態における静脈血と動脈血のpH・pCO2解離、および酸素化評価（P/F比等）におけるVBGの不適合性に関する集中治療医学のコンセンサス。
